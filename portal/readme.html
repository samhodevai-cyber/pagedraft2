# 삼호개발 임직원 복지포털 — 프로젝트 핸드오프 문서 이 문서는 Claude 웹 대화에서 제작한 "삼호개발 복지포털"을 **Claude Code(또는 다른 개발 환경)로 그대로 옮겨 이어 작업**하기 위한 인수인계 자료입니다. 프로젝트 개요, 파일 구성, 기술 구조, 디자인 규칙, 참고 자료(원본 데이터), 제작 히스토리, 남은 작업이 모두 담겨 있습니다. --- ## 1. 프로젝트 개요 - **목적**: 삼호개발 임직원이 복리후생 제도를 한눈에 쉽게 확인할 수 있는 사내 웹 포털 - **형태**: 외부 라이브러리 없이 동작하는 **단일 HTML 파일** (브라우저에서 바로 열림, 인터넷 불필요) - **디자인 방향**: 사내 그룹웨어(MCFrame 기반)와 **통일감** — 그룹웨어에서 링크로 넘어와도 같은 시스템처럼 보이도록 제작 - **주요 기능**: 카테고리 탐색, 키워드 검색, 상세 팝업(모달), 직급별 지원금 표, ERP 신청 가이드 이미지(클릭 확대), 전화 연결 링크 --- ## 2. 파일 구성 Claude Code로 옮길 때 아래 세 파일을 **같은 폴더**에 두세요. | 파일명 | 설명 | |---|---| | `index.html` | 포털 본체 (HTML/CSS/JS 전부 포함). 대화의 "삼호개발 복지포털" 아티팩트를 다운로드해 이 이름으로 저장 | | `annual-leave-guide.png` | 연차 신청 ERP 화면 스크린샷. 휴가제도 → 연차휴가 팝업에서 사용 | | `README.md` | 이 문서 | > `annual-leave-guide.png` 가 없으면 연차휴가 팝업의 이미지 자리에 "이미지 파일을 같은 폴더에 넣어주세요" 안내가 대신 표시됩니다. 대화에 올렸던 ERP 스크린샷을 이 이름으로 저장해 두면 자동으로 나타납니다. --- ## 3. 기술 구조 (Claude Code가 이해해야 할 핵심) 순수 HTML + CSS + 바닐라 JavaScript로만 작성했습니다. 빌드 도구·프레임워크·npm 의존성이 전혀 없습니다. ### 동작 원리 - 화면에 `
` 라는 **빈 컨테이너**를 하나 두고, JavaScript가 상태에 따라 그 안의 HTML을 통째로 갈아 끼우는 **단일 페이지(SPA-lite)** 방식. - 전역 상태: `view`(현재 섹션: home / benefits / leave / allowance / company), `search`(검색어). - `render()` 가 상태를 보고 알맞은 `renderXxx()` 함수를 호출 → 결과 HTML 문자열을 `content.innerHTML` 에 주입. - 상단 가로 메뉴 클릭 → `go(view)` → `view` 변경 → `render()` → `setActiveMenu()` 로 활성 메뉴 강조. ### 주요 함수 - `renderHome / renderBenefits / renderLeave / renderAllowance / renderCompany` : 각 섹션 화면 생성 - `renderSearch()` : 검색 결과 화면 - `openModal(key)` : 상세 팝업 열기 (복지 상세 / 휴가 단계 가이드) - `openImg(src)` : 이미지 클릭 시 전체화면 라이트박스 - `sectionBanner(key)` : 각 섹션 상단의 일러스트 배너(SVG, 인라인) - `svg(name)` : 인라인 SVG 아이콘 생성 (아이콘은 `I` 객체에 path 정의) ### 데이터 위치 (내용 수정은 여기만 고치면 됨) 모든 표시 내용은 코드 상단의 **데이터 배열**에 들어 있습니다. 디자인 코드를 건드리지 않고 이 배열만 수정하면 화면이 자동 갱신됩니다. - `categories` : 홈 바로가기 카드 & 메뉴 순서 기준 - `basicBenefits` : 기본 복지 11종 - `extraBenefits` : 추가 복지·포상 5종 - `education` : 교육 제도 - `leaveTypes` : 휴가 종류별 안내·신청 단계 (연차에는 `img` 필드로 가이드 이미지 연결) - `carTable` / `phoneTable` : 직급별 차량유지비 / 모바일폰 지원금 표 - `searchIndex` : 위 데이터로 자동 구성되는 검색 색인 --- ## 4. 디자인 규칙 (그룹웨어 통일 스타일) CSS 변수(`:root`)로 색상을 관리합니다. | 토큰 | 값 | 용도 | |---|---|---| | 상단 그라데이션 | `#15bfa6 → #2f97ce → #3b7ad9` | 최상단 띠 (그룹웨어와 동일한 청록→파랑) | | `--primary` | `#2f6fd1` | 강조 파랑 (제목 막대, 링크, 활성 메뉴) | | `--menu` | `#6b6c8c` | 상단 메뉴 기본 글자색 | | `--bg` | `#eef2f7` | 페이지 배경 | | `--line` | `#e3e8ef` | 카드/패널 테두리 | | `--ink` | `#3a4252` | 제목 글자색 | 디자인 특징: 상단 그라데이션 바(처음으로/ERP 링크 + 오늘 날짜) → 흰색 메뉴 바(Samho 원형 로고 마크 + 가로 메뉴) → 검색 줄 → 본문. 카드는 모서리 8px·얇은 테두리의 **패널형**, 섹션 제목 앞에는 파란 **세로 막대(│)** 포인트. --- ## 5. 참고 자료 (원본 데이터 출처) 아래 4개 사내 문서를 근거로 제작했으며, 모든 내용은 `index.html`의 데이터 배열에 반영되어 있습니다. 1. **우리회사 복지제도.txt** — 기본 복지 11종, 콘도 지원 대상/범위, 추가 복지·포상, 교육(한마음 교육·직무역량·멘토링) 2. **복리후생 및 연차휴가 상세.txt** — 연차 부여/정산, 미사용 연차 수당 계산, 잔여연차 확인, 연차·반차·경조·병가·휴직 신청 방법, 이사비 서류, 상조용품 3. **삼호개발 세부정보.txt** — 임원, 기업 이념·인재상, 본사/서울사무소 주소·연락처, 건설업 면허, 창립일, 정직원 수 4. **임직원 복리후생비 지원 규정.txt** — 차량유지비·교통보조금, 유류비, 모바일폰 지원금(직급별 표), 이사비 지원 규정 ### 핵심 사실 요약(빠른 참조용) - **연차**: 매년 3/1 부여, 미사용분은 당해 5월 중 수당 지급. 일급 = 통상임금(기본급+직책수당+현장수당) ÷ 209시간 × 8시간. 잔여연차는 ERP `연차신청서_개인`의 잔여일수. - **이사비**: 편도 750,000원 이내 실비(도서지역 대표이사 승인 시 +250,000원). 서류: 주민등록등본(이사 후)·이사비 영수증. 반경 20km(특별시·광역시 30km) 이내 전입, 국내 현장 한정. - **유류비**: 현장 차량당 월 200ℓ(터널 150ℓ) 원칙, 초과분 개인 차량지원금에서 공제. - **차량유지비(현장/본사, 원)**: 임원 200,000/200,000 · 부장·차장 200,000/150,000 · 과장·대리·주임·사원·촉탁 200,000/100,000 - **모바일폰(현장/본사, 원)**: 임원 50,000/50,000 · 부장·차장 50,000/30,000 · 과장 40,000/20,000 · 대리 30,000/- · 주임·사원·촉탁 20,000/- - **본사(당진공장)**: 충남 당진시 면천면 면천로 183 / 041-356-4588 - **서울사무소**: 서울 서초구 효령로 96 삼호빌딩 / 02-2046-7700 / 근무 08:30~18:30, 점심 11:30~12:30 - **문의처**: 복지 신청·규정 = 총무부 02-2046-7712~7715 / 콘도 = 비서실 02-2046-7700 - **기업 이념**: 솔선수범 · 인화단결 · 건설보국 - **연혁**: 1976년 창립, 정직원 338명(25.09.10 기준) --- ## 6. 제작 히스토리 (변경 이력) 1. **초기 제작** — 복지 포털 1차 버전(카테고리 탐색, 검색, 휴가 신청 가이드, 지원금 표, 회사정보). 2. **단일 HTML 파일로 전환** — 자체 호스팅 시 Claude 표시 없이 회사 사이트로 운영 가능하도록, 외부 의존성 없는 단일 파일로 재작성. 3. **섹션 배너 추가** — 복지/휴가/지원금/회사정보 각 섹션 상단에 주제별 인라인 SVG 일러스트 배너 삽입. 4. **버그 수정 ①** — 클릭형 항목 옆 `>` 아이콘이 검은 삼각형으로 표시되던 문제. `.chev`에 `fill:none` 지정으로 해결. 5. **버그 수정 ②** — 클릭형 카드(콘도 지원·교육 등)의 `onclick` 속성에 닫는 따옴표(`"`)가 빠져 HTML이 깨지고 내용이 사라지던 문제. 따옴표 보완으로 해결. 6. **연차 가이드 이미지** — 연차휴가 팝업에 ERP 신청 화면 썸네일 추가, 클릭 시 전체화면 확대(라이트박스). `annual-leave-guide.png` 파일 참조 + 파일 부재 시 안내 표시. 7. **그룹웨어 통일 리디자인** — 상단 그라데이션 바, 흰색 메뉴 바, Samho 브랜드 마크, 패널형 카드, 세로 막대 제목 등으로 전면 개편. 뒤로가기 버튼 대신 상단 고정 가로 메뉴로 전환. 8. **순서·연락처 반영** — 홈 카드 및 상단 메뉴를 회사정보 → 복지제도 → 지원금 → 휴가제도 순으로 정렬. 회사정보 문의처의 '복지 신청·규정 문의'에 총무부 연락처(02-2046-7712~7715) 추가. 홈 섹션 제목을 '안내'로 변경. --- ## 7. 남은 작업 / 주의사항 (TODO) -
