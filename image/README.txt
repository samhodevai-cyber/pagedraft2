이 폴더는 '콘도 관리' 탭에서 업로드한 콘도 카드 이미지(PNG)를 보관하는 곳입니다.

[index8.html부터: 자동 저장]
프로젝트 루트의 serve.ps1 로컬 서버로 페이지를 열면(예: http://localhost:8731/),
'콘도 관리'에서 콘도명을 입력한 뒤 이미지를 업로드할 때 별도 확인 없이 바로
이 폴더에 "콘도명.png" 형식으로 자동 저장됩니다(브라우저 다운로드 다이얼로그 없음).
이는 serve.ps1이 /upload-image 라는 작은 업로드 엔드포인트를 제공해서 가능합니다.

[serve.ps1 없이 html 파일을 직접 열었을 때: 수동 이동 폴백]
정적 HTML을 file://로 직접 열면 브라우저 보안 정책상 웹페이지가 사용자 디스크에
파일을 직접 쓸 수 없습니다. 이 경우 자동 저장이 실패하고 대신:
  1. 이미지를 PNG로 변환해 자동으로 다운로드합니다 (보통 다운로드 폴더에 저장됨).
  2. 다운로드된 PNG 파일을 이 image 폴더로 옮겨주세요.
  3. 파일명은 업로드 시 안내된 이름(예: image/용평 베르데힐.png)과 동일하게
     맞추면, 콘도 정보의 '이미지 URL' 필드에 자동으로 같은 경로가 채워져 있어
     페이지를 새로고침해도 카드에 이미지가 계속 표시됩니다.

대신 정식 서버 API가 같은 역할을 하도록 구현할 수 있습니다.
